# FAQ Chatbot Frontend

Frontend for the RAG-based FAQ assistant for Navi Mutual Fund schemes.

## Features

- Dark theme UI
- Real-time chat interface
- Example questions
- Connected to Hugging Face backend API

## Environment Variables

The following environment variable is configured in the Space settings:

- `NEXT_PUBLIC_API_URL`: Hugging Face backend API URL (default: https://sai68-RAGfaq-chatbot-api.hf.space)

## Access

Visit the Space URL to access the chat interface.
